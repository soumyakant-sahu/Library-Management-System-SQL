# Library Management System (SQL)

## Overview
This project is a Library Management System database designed using MySQL. It demonstrates relational database design using SQL with proper constraints and relationships.

## Features
- Authors table
- Categories table
- Books table
- Primary Keys
- Foreign Keys
- CHECK Constraints
- UNIQUE Constraints

## Technologies Used
- MySQL
- SQL
