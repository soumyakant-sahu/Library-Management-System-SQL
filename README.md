# 📚 Library Management System (SQL)

A complete **Library Management System** built using **MySQL**. This project demonstrates relational database design, data integrity, and advanced SQL concepts such as stored procedures, functions, triggers, views, and transactions.

---

## 🚀 Features

- Manage authors, categories, books, and members
- Issue and return books
- Track overdue books
- Maintain book availability automatically using triggers
- Search books by title, author, or category
- View borrowing history
- Generate reports using SQL queries and views

## 🗄 Database Schema

The database consists of the following tables:

- Authors
- Categories
- Books
- Members
- Book Issues

Relationships are implemented using **Primary Keys** and **Foreign Keys** to ensure referential integrity.
